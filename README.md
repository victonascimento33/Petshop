Projeto front-end desenvolvido para um pet shop fictício, utilizando HTML5 e a estilização com CSS3.O projeto conta com um menu de navegação direto, dividindo o site nas seguintes páginas:

Apresentação:A página inicial, que dá as boas-vindas ao usuário.

Cuidados:Uma área dedicada à apresentação dos serviços aos animais.

Produtos:Uma vitrine digital apresentando os itens oferecidos pela loja.Layout feito por meio de tabelas.

Pedidos:Seção onde os clientes podem fazer solicitações de produtos ou agendar serviços, através do formulário.
